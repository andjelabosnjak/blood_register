<h1>Registar dobrovoljnih darivatelja krvi - BLOOD REGISTER</h1>
<hr>

## Instalacija

```
$ git clone https://github.com/andjelabosnjak/blood_register.git
$ cd blood_register
$ cp .env.example .env and fill your database information
$ composer install
$ php artisan key:generate
$ php artisan migrate
$ php artisan db:seed
```
